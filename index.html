Fake Job Postings Classifier | Hybrid NLP System

Original Research | 2025

🔬 Research contribution: Hybrid fraud detection system combining TF-IDF text analysis with hand-crafted behavioral signals (has_company_logo, urgency_language, text_length_anomalies) — achieving 98.4% accuracy on severely imbalanced data (95/5 split).

98.4% Accuracy
0.988 ROC-AUC
18K+ Job Postings
  • 🆕 Hybrid Feature Architecture: Combines 5,000 TF-IDF bigrams with 9 hand-crafted behavioral anomaly signals (logo presence, urgency markers, length deviations) in a unified sparse matrix — outperforming text-only and numeric-only baselines
  • 🆕 Imbalance Strategy: Stratified 80/20 split preserving real-world 4.84% fraud rate in test set, SMOTE oversampling on training data only — avoiding test contamination while maintaining generalization
  • 🆕 Threshold Optimization Framework: Developed F1-maximization algorithm scanning full precision-recall curve to identify optimal decision boundary (0.50→0.47), achieving 20% F1 improvement over default threshold — methodology applicable to any imbalanced classifier
  • 🆕 Modular Inference Class: Architected JobFraudDetector wrapper supporting 3 inference modes (raw text/batch list/DataFrame) with probability scores, risk levels, and confidence metrics — designed for easy integration into larger applications
  • 🆕 Comparative Model Analysis: Rigorous 5-Fold Stratified CV across 4 algorithms with composite scoring (0.6×F1 + 0.4×ROC-AUC) — Random Forest achieved 0.803 F1 and 0.988 ROC-AUC, establishing new benchmark for job fraud detection
  • 🆕 Behavioral Signal Discovery: Identified that postings without company logos have 15.9% fraud rate vs 2.0% with logos — visual trust signals as key discriminator
